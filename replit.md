# Python Carreiras

## Overview
A Python Flask web application for exploring careers in Python development. The application displays information about various Python-related career paths.

## Project Structure
```
├── main.py              # Main Flask application entry point
├── templates/           # Jinja2 HTML templates
│   ├── index.html       # Home page
│   └── carreiras.html   # Careers listing page
├── static/              # Static assets
│   └── style.css        # CSS styles
├── pyproject.toml       # Python dependencies
└── README.md            # Project readme
```

## Technologies
- **Language**: Python 3.11
- **Framework**: Flask 3.x
- **Templating**: Jinja2
- **Styling**: CSS3

## Running the Application
The application runs on port 5000 and is accessible via the Replit webview.

```bash
python main.py
```

## Routes
- `/` - Home page with hero section
- `/carreiras` - Careers listing page
